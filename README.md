# Circuitos Digitais

Esse repositório contém os laboratórios desenvolvidos na disciplina de circuitos
digitais, bem como seu projeto final: Neander.

## Do Neander
Neander é um processador teórico cuja ISA foi utilizada na disciplina de Introdução à
Arquitetura de Computadores.

## Parte prática da disciplina em questão
Para Circuitos Digitais, será feita a implementação
da CPU em nível de portas lógicas. Todos os laboratórios
da disciplina são uma espécie de guia para esse fim.
